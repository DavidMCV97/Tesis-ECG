# Tesis Elecrocardiogramas
Proyecto de tesis enfocado en implementar herramientas de ML en el
análisis automático de electrocardiogramas.

## Herramientas
- Análisis estadístico
  - Correlación
  - PCA
  - Medidas de tendencia central
  - Medidas de variación
- Visualización
  - Boxplots
  - Colormap de correlación
  - Histogramas
- Machine Learning
  - Label Encoder
  - Train Test Split
  - Random Forest
  - Gridsearch
  - Cross Validation
## Contenido
- AED: Archivos de análisis exploratorio de datos sobre los elecrocardiogramas.
- Random Forest: Archivos de experimentación e implementación final del algoritmo Random Forest.
- Train Test Split: Archivo generador del split único para usar el mismo split en todos los experimentos.
